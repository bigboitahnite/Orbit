# 🌑 Orbit

A fast, dark web proxy with a clean lunar aesthetic. Built on top of [Ultraviolet](https://github.com/titaniumnetwork-dev/Ultraviolet) and [Bare Server](https://github.com/nicholasstephan/bare-server-node).

---

## Features

- 🌑 Dark moon-themed UI
- 🔍 Proxy any URL via Ultraviolet
- 🎮 50+ unblocked games
- 📱 Unblocked apps
- ⚡ Tab cloaking
- 🔲 About:Blank mode
- 🚀 One-click deploy to Render

---

## Setup

### Requirements
- Node.js v16+
- npm / pnpm / bun

### Run Locally

```bash
git clone https://github.com/YOUR_USERNAME/orbit
cd orbit
npm install
npm start
```

Then open `http://localhost:8080`

---

## Deploy to Render

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy)

1. Fork this repo
2. Go to [render.com](https://render.com) → New Web Service
3. Connect your GitHub repo
4. Build command: `npm install`
5. Start command: `npm start`
6. Done ✅
